Maker Coin - created in OpenSCAD

versions
	Makercoin - main one - just a coin
	RFID - hollow so you can pause a print and drop a RFID sticker inside
	Keyring - hole to put on a chain
	RFID Keyring - both of the above

3mf files are precoloured and work in prusa slicer and bambu labs
stl - each colour part has been exported separately, follow version naming if a colour has it to get the right model
	i,e black rfid part is the same across all versions
